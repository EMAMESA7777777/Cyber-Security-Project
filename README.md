Practical lab exercises and reports completed during a Penetration Testing & Cybersecurity course using Kali Linux and industry-standard security tools.

## 📚 Topics Covered

- Web Reconnaissance
- Network Enumeration
- Metasploit Framework
- Hydra (Brute-force Testing)
- Wireshark Traffic Analysis
- Password Hash Analysis
- ARP & FTP Traffic Analysis

## 🛠 Tools Used

| Tool | Purpose |
|------|---------|
| Nmap | Network discovery & port scanning |
| Metasploit | Exploitation framework |
| Hydra | Brute-force credential testing |

| John the Ripper | Password hash cracking |
| Kali Linux | Primary penetration testing OS |
